📚 Colab Documentation Generator

🚀 Colab Documentation Generator is an AI-powered tool that automatically generates comprehensive, well-organized documentation from Jupyter Notebooks (Colab Notebooks). The project leverages advanced AI models to analyze the code and markdown cells, producing professional-quality documentation that is clean, readable, and suitable for knowledge transfer or project handover.

🌟 Features
Automatic Documentation Generation: Generates clean, structured, and professional documentation from Colab notebooks.

Multi-File Support: Upload multiple notebooks simultaneously and get consolidated documentation.

Language Support: Generate documentation in multiple languages, including English, Spanish, French, German, Chinese, Hindi, and Japanese.

PDF Export: Download the generated documentation as a professional PDF.

User-Friendly Interface: Intuitive UI for uploading files, selecting languages, and downloading documentation.

Handles Large Files: Configurable file size limits to support larger notebook uploads.

Efficient and Scalable: Uses OpenRouter AI API for fast and accurate documentation generation.

🚀 Technology Stack

Frontend:

React.js with TailwindCSS for UI

Axios for API requests

Backend:

Node.js with Express.js

OpenRouter AI API for generating documentation

Puppeteer for PDF generation

Python (for notebook processing)

Pandas and nbformat for parsing notebooks
